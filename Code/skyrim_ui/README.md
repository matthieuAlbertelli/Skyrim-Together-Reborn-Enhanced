# Skyrim Together UI

This Angular application provides the in-game CEF interface used by Skyrim Together Reborn and STRE features such as player trading.

## Install dependencies

```powershell
pnpm install
```

## Development server

```powershell
pnpm start
```

## Production build

```powershell
pnpm deploy:production
```

Output is written to `dist/`. Do not commit `node_modules`, `.angular` caches or generated deployment copies.

For the full native + UI workflow, see [`../../docs/development/BUILDING.md`](../../docs/development/BUILDING.md). The upstream build guide is also referenced there.
