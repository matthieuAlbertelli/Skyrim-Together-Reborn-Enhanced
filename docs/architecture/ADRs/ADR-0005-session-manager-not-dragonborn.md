# ADR-0005 — Session Manager et Dragonborn sont deux rôles distincts

- **Statut : Accepted**
- **Date : 2026-07-19**

## Contexte

L’hôte technique ne doit pas devenir automatiquement le héros canonique.

## Décision

Le Session Manager administre la campagne. Le Dragonborn est un rôle narratif assigné et révélé selon les règles de campagne.

## Conséquences

- narration plus juste pour le groupe ;
- policies d’autorité explicites ;
- gestion du Dragonborn absent à définir ;
- secret filtré côté serveur.
